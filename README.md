# aliens_simulation

Descrição
Breve simulação/projeto chamado "aliens_simulation". Este README contém instruções básicas para instalar dependências e executar o projeto. Ajuste os comandos abaixo conforme o entrypoint real do seu repositório.

Pré-requisitos

- Git
- Node.js 14+ / npm

Clonar repositório

1. git clone <URL_DO_REPOSITORIO>
2. cd aliens_simulation

Node.js

1. Instalar dependências:
   - npm install
   - ou alternativa: npm i
2. Executar:
   - Se houver script no package.json: `npm start`
   - Ou diretamente: `node index.js` ou `node src/index.js` (substitua conforme necessário)

Testes

- Node: `npm test`
